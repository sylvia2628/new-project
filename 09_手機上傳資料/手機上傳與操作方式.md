# 手機上傳與操作方式

## 重要觀念

目前 `慕舍室內設計` 資料夾是在你的 Mac 電腦裡，不是雲端系統，也不是已上線的平台。

所以手機不能自動把資料寫進這個資料夾，除非你使用以下其中一種方式：

1. 直接在手機上把資料傳到這個 AI 對話
2. 用 iCloud Drive / Google Drive / Dropbox 這類雲端同步
3. 用 AirDrop 傳到 Mac，再拖進資料夾
4. 用 LINE / Email 先暫存，再從 Mac 下載整理

第一階段建議用最簡單的方法：手機把資料傳給 AI，AI 再協助整理進專案資料夾。

## 目前可用的 GPT 遠端連線方式

第一版的「手機 GPT 遠端連線」不是手機直接連到 Mac 資料夾，而是用同一個 GPT / Codex 對話當橋接：

```text
手機 ChatGPT App 或手機瀏覽器
  ↓
同一個 GPT / Codex 對話
  ↓
AI 判斷資料類型並整理
  ↓
Mac 本機專案資料夾
  ↓
MUSE AI OS 原型與資料文件
```

這樣做的好處是第一版不需要架設伺服器、不需要帳號密碼，也不需要先串接 LINE、Google Drive 或 OpenAI API。

### 手機端操作

1. 用手機打開目前這個 GPT 對話
2. 上傳截圖、照片，或直接貼上語音轉文字
3. 加上一句標準指令，讓 AI 知道要整理到哪個模組
4. 等 AI 回覆整理結果

### Mac 端整理

AI 會依資料類型整理到：

- 行程：`09_手機上傳資料/01_行事曆截圖`
- LINE 對話：`09_手機上傳資料/02_LINE對話截圖`
- 客戶資料：`09_手機上傳資料/03_客戶資料`
- 案件進度：`09_手機上傳資料/04_案件進度`
- 作品素材：`09_手機上傳資料/05_作品素材`
- 不確定分類：`09_手機上傳資料/99_待整理`

### 手機 GPT 可貼指令

行程整理：

```text
這是今天或本週行程，請幫我整理到 MUSE AI OS：今日行程、本週提醒、需要追蹤的客戶、需要新增的工作。
```

客戶對話整理：

```text
這是客戶 LINE 對話，請幫我整理：客戶需求摘要、尚缺資料、下一步追蹤、CRM 欄位、LINE 回覆草稿。不要直接發送。
```

案件進度更新：

```text
這是案件進度更新，請整理：案件名稱、目前階段、完成百分比、下一步、截止日、負責人、是否有延誤風險。
```

作品素材整理：

```text
這是作品素材，請幫我整理：作品名稱、空間類型、風格、可公開素材、缺少素材、官網案例文案、IG 文案與 Google 商家文章草稿。
```

## 目前不能直接做到的事

第一版仍然不能做到：

- 手機直接自動寫入 Mac 本機資料夾
- 手機直接更新 `08_AI小助手版面/index.html`
- AI 未經確認直接發送 LINE 或社群貼文
- 未授權就讀取 TimeTree、LINE、Google Drive、Instagram 或 Facebook

如果之後要做到全自動同步，需要正式建立後端系統，並使用官方 API / OAuth 授權。

## 已建立的手機資料收件夾

手機傳來的資料可先放在：

```text
慕舍室內設計/09_手機上傳資料
```

子資料夾：

```text
01_行事曆截圖
02_LINE對話截圖
03_客戶資料
04_案件進度
05_作品素材
99_待整理
```

## 方法一：最快方式，直接傳到 AI 對話

適合：

- TimeTree 截圖
- LINE 對話截圖
- 客戶需求截圖
- 口述轉文字
- 臨時工作事項

操作方式：

1. 用手機打開這個 AI 對話
2. 上傳截圖，或直接貼文字
3. 加上一句你要 AI 做什麼

範例：

```text
這是今天 TimeTree 行程截圖，請幫我整理到 MUSE AI OS 的行程資料。
```

或：

```text
這是客戶 LINE 對話，請幫我整理客戶摘要、尚缺資料、下一步追蹤與回覆草稿。
```

優點：

- 最快
- 不需要先會整理檔案
- AI 可以直接幫你判斷要放到行程、工作、客戶還是案件

注意：

- 截圖要清楚
- 日期、時間、客戶名稱要看得到
- 如果截圖中有多個案件，最好補一句說明

## 方法二：iPhone 檔案 App + iCloud Drive

適合：

- 你希望手機可以直接把檔案放進類似資料夾的地方
- 你習慣用 iPhone 的「檔案」App

前提：

- Mac 和 iPhone 使用同一個 Apple ID
- Mac 已開啟 iCloud Drive
- 若要在 iPhone 看到桌面資料夾，Mac 需開啟「桌面與文件夾」同步

操作方式：

1. iPhone 打開「檔案」App
2. 進入「iCloud Drive」
3. 找到 `Desktop` 或你設定的同步資料夾
4. 進入：

```text
Codex/慕舍室內設計/09_手機上傳資料
```

5. 把截圖、照片或文字檔放到對應資料夾

建議放法：

- TimeTree 截圖：放 `01_行事曆截圖`
- LINE 對話：放 `02_LINE對話截圖`
- 客戶資料：放 `03_客戶資料`
- 案件進度：放 `04_案件進度`
- 作品照片影片：放 `05_作品素材`
- 不確定分類：放 `99_待整理`

注意：

如果 iPhone 的檔案 App 看不到 Mac 桌面，代表 Mac 沒有開啟桌面同步。這時先用方法一、三或四會比較快。

## 方法三：AirDrop 傳到 Mac

適合：

- 傳 TimeTree 截圖
- 傳現場照片
- 傳作品照片
- 傳 LINE 截圖

操作方式：

1. iPhone 選取照片或截圖
2. 按分享
3. 選 AirDrop
4. 傳到你的 Mac
5. Mac 接收後，通常會在「下載項目」
6. 再把檔案拖到：

```text
慕舍室內設計/09_手機上傳資料/99_待整理
```

或依類別拖到對應資料夾。

## 方法四：Google Drive 暫存

適合：

- 團隊多人也要上傳資料
- 之後可能要接 Google Drive API
- 作品照片與施工圖檔案較多

建議建立一個 Google Drive 資料夾：

```text
MUSE AI OS 手機上傳
```

下面再建立：

```text
01_行事曆截圖
02_LINE對話截圖
03_客戶資料
04_案件進度
05_作品素材
99_待整理
```

操作方式：

1. 手機打開 Google Drive App
2. 上傳照片、截圖或檔案
3. 放到對應資料夾
4. 之後在 Mac 下載或同步到本機專案資料夾

注意：

第一階段不需要提供 Google 帳號密碼。未來若要自動同步，應使用 Google Drive API 授權。

## 方法五：LINE 暫存

適合：

- 你日常最常用 LINE
- 想先快速把資料丟到某個地方

建議做法：

1. 建一個只有自己或內部人員的 LINE 群組
2. 群組名稱可用：

```text
MUSE AI OS 資料暫存
```

3. 把 TimeTree 截圖、客戶截圖、工作事項丟進去
4. 在 Mac 上打開 LINE，下載後放入 `09_手機上傳資料`

建議每次丟資料時加一行說明：

```text
類型：行程
日期：2026-07-13
案件：林先生鼓山新成屋
要做：整理成今日行程
```

## 建議檔名格式

若你可以改檔名，建議使用：

```text
日期_類型_案件或客戶_說明
```

範例：

```text
2026-07-13_行程_TimeTree本週截圖.png
2026-07-13_LINE_林先生鼓山新成屋_初次詢問.png
2026-07-14_案件進度_新文化城安_提案調整.txt
2026-07-16_作品素材_微笑城堡_完工照01.jpg
```

如果不會改檔名也沒關係，可以先全部放到 `99_待整理`。

## 每天最簡單的手機工作流程

### 早上

把 TimeTree 今日或本週截圖傳給 AI，並說：

```text
請幫我整理今天的行程、提醒與優先順序。
```

### 工作中

有新工作就用語音輸入：

```text
新增工作：聯上康橋施工圖，這週三前完成，指派設計師，優先程度高。
```

### 客戶詢問

把 LINE 對話截圖或文字傳給 AI：

```text
請幫我整理這位客戶的需求摘要，並產生一版 LINE 回覆草稿。
```

### 下班前

用文字或語音補一句：

```text
請幫我整理今天完成事項、未完成事項、明天提醒。
```

## 隱私提醒

客戶電話、地址、預算與合約屬於敏感資料。

如果只是要測試流程，可以先遮蔽電話、地址或姓名。

如果是正式整理公司資料，建議資料夾只放在公司可控的 Mac、公司 Google Drive 或授權帳號中。
