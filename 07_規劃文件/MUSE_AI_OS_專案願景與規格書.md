# MUSE AI OS 專案願景與規格書

## 專案願景

我們希望打造的不只是管理系統，而是一位真正了解室內設計流程的 AI 秘書。它能主動整理工作、提醒進度、協助團隊合作、管理客戶與作品，減少行政工作，讓設計師能把更多時間投入設計與服務客戶。

平台將採模組化設計，先完成核心管理功能，再逐步串接 LINE 官方帳號、Facebook、Instagram、Google Drive、TimeTree 等工具，成為慕舍室內設計的 AI 營運中心。

## 一、專案目標

建立一套專屬於慕舍室內設計公司的 AI 管理平台：MUSE AI OS。

本平台不是對外官網，而是公司內部使用的 AI 工作中心，目標是：

- 降低行政工作時間
- 統整所有案場資訊
- 建立 AI 秘書
- 提升團隊協作效率
- 自動整理客戶資料
- 管理工作進度
- 管理作品資料
- 整合各種工作平台

## 二、使用者

平台採多人共同使用，每位使用者需有獨立帳號及不同權限。

角色包含：

- 公司負責人
- 室內設計師
- 設計助理
- 工務
- 財務
- 行政
- 未來：工班

## 三、平台定位

本平台不是：

- 官網
- LINE 官方帳號
- Facebook
- Instagram

本平台是公司內部 AI 工作中心。所有社群平台僅作為對外入口，AI 平台負責管理所有資料。

## 四、主要模組

### 1. AI 秘書

首頁每天主動提醒：

- 今日工作
- 本週工作
- 待回覆客戶
- 即將提案
- 即將丈量
- 即將完工
- 即將收款
- 即將付款
- 待上架作品

AI 不需等待詢問，每天主動提供工作摘要。

### 2. 客戶管理 CRM

建立完整客戶資料：

- 姓名
- 電話
- LINE
- Email
- 地址
- 案場
- 坪數
- 預算
- 喜好風格
- 家庭成員
- 特殊需求

AI 可以：

- 整理聊天紀錄
- 建立客戶摘要
- 提醒多久未聯絡
- 產生追蹤訊息

### 3. 案場管理

每個案件建立完整流程：

- 初步諮詢
- 丈量
- 平面配置
- 3D
- 施工圖
- 估價
- 合約
- 工程
- 驗收
- 完工

每個案件顯示：

- 完成百分比
- 負責人
- 截止日期
- 下一步工作
- 是否延誤

### 4. 行程管理

目前公司使用 TimeTree。

第一階段不更換 TimeTree。AI 協助整理：

- 今日行程
- 本週行程
- 工作提醒

未來可研究：

- TimeTree 同步
- Google Calendar 同步

### 5. 工作管理

建立工作清單，例如：

- 聯上康橋施工圖
- 新文化城安提案
- 微笑城堡拍攝

工作可：

- 指派
- 設截止日期
- 標示完成狀態
- 由 AI 提醒

### 6. 作品管理

每個作品建立：

- 案場照片
- 施工前
- 施工後
- CAD
- 平面圖
- 影片
- 設計理念

AI 自動產生：

- 官網案例
- Facebook 文案
- Instagram 文案
- Google 商家文章
- 短影音腳本

所有內容由管理者確認後發布。

### 7. AI 客戶回覆

AI 可依公司既有回覆方式，自動生成：

- 初次詢問
- 報價說明
- 丈量提醒
- 提案通知
- 工程通知
- 驗收通知
- 尾款通知

所有內容需管理者確認後才發送。

### 8. Google Drive 文件管理

自動整理：

- CAD
- PDF
- 照片
- 合約
- 報價
- 施工圖

AI 可依案件分類。

### 9. 財務管理：第二階段

整合：

- Excel
- 工程支出
- 收款
- 尾款
- 毛利分析
- 工班支出

## 五、AI 功能

AI 可：

- 整理會議內容
- 整理 LINE 對話
- 建立工作摘要
- 每日工作報告
- 每週工作報告
- 提醒重要事項
- 整理作品
- 自動產生文案

## 六、權限管理

依角色設定：

- 管理者：所有資料
- 設計師：自己案件
- 助理：工作、行程、客戶
- 工務：工程
- 會計：財務

## 七、未來串接

第一階段不需要提供任何帳號密碼。

未來使用官方 API：

- LINE Official Account API
- Facebook API
- Instagram API
- Google Calendar API
- Google Drive API
- Gmail API

避免提供登入帳號與密碼，提高安全性。

## 八、平台特色

MUSE AI OS 不是一般 ERP。

它是：

- AI 室內設計秘書
- CRM
- 案場管理
- AI 行銷中心
- 團隊協作平台

目標是讓 AI 主動協助公司運作，而不是單純回答問題。

## 九、第一版 MVP

優先完成以下五大功能：

1. AI 秘書首頁：每日工作摘要
2. 客戶 CRM 管理
3. 案場進度管理
4. 工作 / 待辦事項管理
5. 作品管理，含 AI 文案

待第一版穩定後，再逐步加入：

- TimeTree 整合
- Google Drive
- 財務
- LINE 官方帳號
- Facebook
- Instagram
- 官網串接
