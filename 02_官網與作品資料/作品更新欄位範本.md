# 作品更新欄位範本

新增作品時，建議每個作品建立一個資料夾，並放入照片、平面圖、作品說明與是否可公開的標記。

## 基本欄位

- 案名：
- 地區：
- 空間類型：住宅 / 商業空間 / 老屋翻新 / 預售屋 / 接待中心 / 銷售中心
- 屋況：新成屋 / 中古屋 / 老屋 / 預售屋 / 其他
- 坪數：
- 格局：
- 設計風格：
- 完工年份：
- 主要需求：
- 設計重點：
- 使用建材：
- 是否可公開：是 / 否 / 需局部遮蔽

## 圖片欄位

- 封面圖：
- 完工照：
- 施工前照片：
- 施工中照片：
- 平面圖：
- 3D 模擬圖：
- 需避開公開的照片：

## 官網文案欄位

- SEO 標題：
- SEO 描述：
- 作品摘要：
- 作品完整文案：
- 圖片 alt 文字：
- 對應服務標籤：

## AI 可協助的任務

- 依欄位產生作品文案
- 依照片與作品資料產生社群貼文草稿
- 產生官網 SEO 標題與描述
- 依作品風格分類
- 轉成 LINE 可回覆的作品推薦話術
